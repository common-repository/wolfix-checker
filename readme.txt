The plugin will send data to Wolfix Web Development company in order to help maintain your site health.

== Description ==

Install the plugin and contact Wolfix Web Design staff. This plugin will send them site health informations and help them fix your website problems.

== Third-Party Services ==

This plugin relies on external third-party services under certain conditions:
- **Google Tag Manager**: Used for site analytics. [Google's privacy policy](https://policies.google.com/privacy)
- **WordPress.org API**: Checks for plugin and theme updates. [WordPress.org privacy policy](https://wordpress.org/about/privacy/)

By using this plugin, you acknowledge that some data will be sent to these services and used in accordance with their respective privacy policies.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
